# 🚀 EduSync Lite – Mediated Real-Time Collaborative Platform

EduSync Lite is a real-time collaborative web application that enables structured knowledge sharing under a mediator-controlled system. It combines real-time updates with role-based access control to ensure organized and secure collaboration.

---

## 🧠 Overview

Unlike traditional open collaboration tools, EduSync Lite introduces a **mediator-based workflow** where all user-generated content (notes, resources) is reviewed before being published. This ensures quality, prevents misuse, and maintains a structured environment.

---

## ✨ Features

- 🔐 **Authentication & Authorization (JWT)**
- ⚡ **Real-Time Updates (Socket.IO)**
- 📝 **Collaborative Notes System**
- 📂 **Resource Upload & Management**
- 🛡️ **Mediator Approval Workflow**
- 📊 **Activity Tracking System**
- 👥 **Role-Based Access Control (RBAC)**
