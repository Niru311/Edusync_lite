Micro-Interactions to Include

To make the UI feel alive:

card lift on hover

animated leaderboard update

XP pop when points earned

confetti on quiz win

smooth carousel sliding

These small details make the platform feel game-like rather than academic.

Final Visual Identity (Best Choice)

The strongest design direction is a hybrid of two themes:

Neon Study Arena
+
Playful Gradient Learning UI

This gives:

gamified energy

colorful gradients

competitive leaderboard feel

friendly student interface

It will look like a modern EdTech startup product, not a static college project.
